# CSharpIDE
 Среда разработки C# для Windows с открытым исходным кодом
## Функции
- [x] Подсветка синтаксиса
- [x] Запуск и компиляция
    - [ ] Отладка
    - [x] Запуск
    - [ ] Компиляция
        - [x] EXE файлы
        - [ ] DLL файлы
- [ ] Дизайн
    - [ ] Вкладки
    - [ ] Встроенный проводник
- [ ] Создание приложений с пользовательским интерфейсом
    - [ ] WPF
    - [ ] WINFORMS
- [ ] Настройки
- [ ] Git
    - [ ] Консоль
    - [ ] Графичечкий интерфейс
- [ ] Консоль
